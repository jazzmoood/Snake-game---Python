
1. **Game Controls:**
- Use arrow keys (`↑`, `↓`, `←`, `→`) to control the snake's direction.
- The objective is to eat as much food (`*`) as possible without colliding into walls or the snake's own tail.

2. **Rules:**
- The snake grows longer as it eats food.
- The game ends if the snake runs into the screen edge or itself.

## Features

- Simple and intuitive gameplay.
- Uses Turtle graphics for visual representation.
- Keeps track of the score (length of the snake).
